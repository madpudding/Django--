"#PrimaryCrawlerWeb"
 
使用Django标准文件框架来构造的整web系统
1. python后台，Django 2.1.7版本，其中使用了Django中2.x版本后path用法。
2. 前端web使用了bootstrap、js、jquery以及bootstrap的组件，页面的逻辑是js、jquery混合编写，其中js使用了部分es6的语法。
3. 使用了后台和前端两种处理数据方式。
